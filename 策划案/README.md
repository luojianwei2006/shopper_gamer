## 如何阅读

1. 先看 `策划案_合成小店长.md` 建立全貌（设计支柱 P1–P6 与核心循环是总纲）。
2. 经济数值与平衡对照 `经济系统平衡表.md`；所有数值为假设值，标 `[PLACEHOLDER]` 者待纸面模拟与投放调优。
3. 交互与流程以 `界面原型_合成小店长.html` 为准（浏览器打开即可点），`界面原型设计说明.md` 解释设计决策。
4. 资源与工程底座分别见 `免费资源清单_Fab.md` 与 `免费UE5框架参考.md`。
